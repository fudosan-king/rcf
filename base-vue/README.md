# admin-ecommerce
### Init env
```
cp .env.example .env
```

### Setup API endpoint in .env
```
VUE_APP_API_ENDPOINT=
```

### Project setup
```
cd src
yarn install
```
### Compiles and hot-reloads for development
```
[***/rcf/src]
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Lints and fixes files
```
yarn run lint
```
